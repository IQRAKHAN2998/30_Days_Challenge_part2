Purpose

The Main Orchestrator Agent acts as the central controller that manages all Sub-Agents. It assigns tasks, collects outputs, ensures workflow order, and maintains consistency across the entire writing or research process.

**Capabilities**


Distributes tasks to the correct Sub-Agents

Ensures logical flow between research, writing, editing, and formatting

Merges results from multiple Sub-Agents

Resolves conflicts and maintains overall project direction

Communication / Interaction

Receives input from the user

Sends research tasks to the Research Sub-Agent

Sends writing tasks to the Writing Sub-Agent

Sends refinement tasks to the Editing Sub-Agent

Sends structural and layout tasks to the Formatting Sub-Agent

Collects all responses and provides a final output to the user
